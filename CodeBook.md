Variable list and descriptions
------------------------------

Variable name    | Description
-----------------|------------
X          	| data table of "mean" or "std" of sensor measurements 
y				| data table of corresponding activity to the sensor measurements
subject 		| data table of corresponding volunteer generating the data
activityLabels 	| mapping table of activityIndex & their Names
mergedDataset 	| tidy dataset merged from "subject", "y", and "X"
groupSummary 		| summary of all variables (mean) by group "subject" & "activity"
